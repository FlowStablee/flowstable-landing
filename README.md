## FlowStable Landing

Minimal, black‑and‑white marketing page for the FlowStable project.

### Tech

- Static `index.html` with inline CSS
- Responsive layout tuned for desktop and mobile

### Local development

You can open `index.html` directly in a browser, or serve the folder with any static file server, for example:

```bash
npx serve .
```

### Deployment

Host the contents of this repository on any static host (such as your existing `flowstable.in` setup, GitHub Pages, Vercel, or Netlify). Point your domain at the deployed build so `https://flowstable.in` serves `index.html`.
